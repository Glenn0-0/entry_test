# ==== TEST ASSIGNMENT ====

How to run

Method 1:
- Download ```game.rar```
- Run ```main.exe```

Method 2:
- Download folder ```src```
- If [pygame](https://www.pygame.org/wiki/GettingStarted) not installed - ```pip install pygame```
- Run ```main.py```

Controls

- ```  ↑  ``` - Jump.
- ```  ←  ``` - Move left.
- ```  →  ``` - Move right.
- ```SPACE``` - Shoot.
